직무능력사항

펼쳐보기
HTML / CSS3
웹표준과 웹접근성을 준수한 시맨틱 마크업을 통해 디자인된 시안을 구현할 수 있으며, 각 요소의 스타일을 파악하여 유지 및 보수가 가능합니다.
JavaScript
사이트를 구현하는데 필요한 다양한 동적 기능과 이벤트를 구현할 수 있습니다.
재사용과 유지보수를 위한 JavaScript module 등을 사용할 수 있습니다.
JQuery
JQuery 를 이용한 다양한 동적 기능과 이벤트를 구현할 수 있습니다.
JSON / AJAX
JSON 을 작성하여 클라이언트와 서버 간의 데이터 교환을 원활하게 수행할 수 있습니다.
AJAX 를 활용하여 비동기적으로 데이터를 요청하고 서버로부터 데이터를 받아오는 웹 페이지의 동적 기능을 구현할 수 있습니다. 이를 통해 사용자 경험을 향상시키고 웹사이트의 성능을 최적화할 수 있습니다.
Git / Git hub
Git을 통해 소스 코드의 버전 관리를 효과적으로 수행할 수 있으며, 원격저장소 git hub를 사용하여 협업이 가능합니다. https://github.com/yieunbin/portfolio.git
VSCode
Visual Studio Code를 활용하여 다양한 프로그래밍 언어를 지원하는 통합 개발 환경에서 효율적으로 코딩 작업을 수행할 수 있습니다.
BootStrap
JavaScript 반응형 프레임워크 중 하나인 bootstrap을 사용하여 태그형 CSS를 사용할 수 있고, 해상도에 따른 최적화된 반응형 웹사이트의 구축이 가능합니다.
Tailwind
Tailwind를 사용하여 태그형 CSS를 활용하고, 해상도에 따른 최적화된 반응형 웹사이트를 구축할 수 있습니다.
WordPress
다양한 테마와 플러그/위젯을 제공하는 오픈소스 CMS 중 가장 많이 쓰이는 WordPress를 이용하여 홈페이지의 구축 및 관리가 가능합니다.
Adobe Photoshop, Illustrator
웹페이지 디자인 작업을 수행할 수 있습니다. 이미지 편집 및 조작, 레이어 기반 디자인 작업을 통해 웹사이트의 시각적 요소를 디자인하고 최적화할 수 있습니다.
Figma
Figma를 사용하여 협업 환경에서 웹페이지 디자인 작업을 손쉽게 진행할 수 있습니다. 기능적 부분에 대한 작업을 효율적으로 관리하며, 반응형 웹사이트의 설계와 프로토타입을 구축하는 데 유용하게 사용할 수 있습니다.
MS Office
Word, Excel, PowerPoint를 사용하여 업무 문서 작성 및 관리가 가능합니다.
Cordova
Cordova를 사용하여 모바일 웹을 하이브리드 앱으로 제작할 수 있습니다.
